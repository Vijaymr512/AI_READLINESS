## Scoring Model Update (recommended)

### Step 1: Base category score
- Per category: `matched_weight / total_weight * 100`

### Step 2: Mandatory controls gate
- Define 20 mandatory controls (security/auth/api/data/logging/ci).
- `mandatory_pass_rate = passed_mandatory / total_mandatory`

### Step 3: Risk penalties
- Critical: `-12` each
- High: `-6` each
- Medium: `-3` each
- Low: `-1` each

### Step 4: Supply-chain confidence bonus
- SLSA + SBOM + signed release + branch protection + code review
- Bonus `+0` to `+10`

### Step 5: Final score
- `final = weighted_category_score + bonus - penalties`
- Clamp: `0..100`

### Step 6: Critical override
- If any critical in `authz`, `crypto`, `secrets`, or `ssrf`: cap at `59`

### Step 7: 80+ readiness gate
- Score >= 80
- Mandatory pass rate >= 0.85
- Critical count = 0
- High count <= 2
